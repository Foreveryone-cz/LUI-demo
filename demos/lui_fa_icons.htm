FontAwesome Icons

.fa_b_[icon-name] | .fa_a_[icon-name]

Using FontAwesome icons for :before/:after: Works only with pair elements. Wont work with input, use button instead

class="fa_b_eye"

icon before 

class="fa_a_eye"

icon after

List of available icons can be found on FontAwesome official website: fontawesome.io/icons/

Usage is simple: fa_b_name or fa_a_name
